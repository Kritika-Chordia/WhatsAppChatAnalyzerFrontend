How to Use:

The steps to be followed to know the detailed analysis of your chats are:

Let's make the conversations more optimized.

  1. Decide the chat on which analysis is to be done.
  2. Export the chat:
    1. Open WhatsApp Application.
    2. Go to group/individual chat and open it.
    3. Tap/Click on the ":" feature in the top right corner.
    4. Tap/Click on the more option.
    5. Now click on export chat option.
    6. Select without media.
    7. Send it to anybody via: email, telegram, whatsApp, drive, copy to device etc.
    8. Finally, we got "chat_name.txt" file.
  3. Use this txt file for analysis.
  4. From Navigation Bar select "Let's Analyse" option.
  5. Upload your ".txt" file in file upload option.
  6. Now if it is a group chat select particular person with whom you want to analyse your chat or select overall to analyse whole group at a time.
